Welcome

Copilot-LD is an intelligent agent leveraging GitHub Copilot, linked data and retrieval-augmented generation.

🎯 Amazingly Accurate

Chunking and similarity searches are done with linked data for amazing accuracy.

⚡️ Incredibly Fast

Retrieval of vectors and text are done in parallel using in-memory indices for incredible speed.

🛡️ Robustly Secure

The system is deployed with network isolation, gRPC, HMAC authentication and a container surface below 10 MB for robust security.

✨ Elegantly Simple

Developed in plain JavaScript with no external dependencies, making the system ridiculously simple to deploy.
